# Rebrand-Blizzard-App

Changes the Blizzard App icons back to Battle.net icons.

Usage
=====

1. Download and extract.
2. Close Blizzard/Battle.net Completely (from tray).

3. Run "RUN_ME.bat"
4. Start Battle.net
